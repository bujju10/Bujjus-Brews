# 🍷 Bujju's Brews | Smart Kitchen & Recipe Manager

**Bujju's Brews** is a high-performance, single-page web application designed for modern kitchen management, precise recipe scaling, and home fermentation tracking. Featuring a sleek, dark-mode glassmorphism interface, it integrates generative AI to empower home chefs and brewers with dynamic culinary tools.

## 🚀 Key Features

* **🤖 AI Recipe Synthesis:** Leverages the Google Gemini AI core to generate highly practical, step-by-step recipes based on real-time inventory inputs.
* **⚖️ Algorithmic Scaler:** A precision culinary calculator that mathematically adjusts ingredient ratios for any target batch size or yield.
* **🛒 Smart Extraction Tools:** Automatically parses AI-generated recipes to create interactive shopping lists and downloadable `.txt` log files.
* **🕰️ Synthesis History:** Automatically saves past AI generations to local storage, allowing for quick retrieval without re-querying the API.
* **📡 Community Broadcast Feed:** A local network feed where users can log and share their successful dishes and brewing timelines.
* **🆔 Local Authentication:** A lightweight, browser-based ID system to sign community posts and maintain user session context.

## 🛠️ Technology Stack

* **Frontend:** Vanilla HTML5, CSS3 (Advanced Glassmorphism & Keyframe Animations)
* **Logic:** JavaScript (ES6+)
* **AI Integration:** Google Gemini 1.5 Flash API
* **Icons & Typography:** FontAwesome 6.4.0, Space Grotesk, Outfit
* **Storage:** Web Storage API (`localStorage`)
